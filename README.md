## Running the App

Clone the repo

```shell
git clone git@github.com:appeltje-c/voice-command-music-search.git
```

Then install and run

```shell
cd voice-command-music-search && yarn && yarn dev
```

An visit [http://localhost:5173](http://localhost:5173)
